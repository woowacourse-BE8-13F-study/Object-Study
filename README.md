# 오브젝트 북스터디
> **객체지향의 사실과 오해, 오브젝트 도서를 기반으로 객체 지향적 설계를 학습하는 스터디**

## 스터디 규칙
### 1. 스터디 시간
- 26/03/16 ~ 26/04/30
- 매주 월요일 10:30 회의실 대면 스터디

### 2. 스터디 진행 방향
- 매주 10분정도 2명이 학습 정리한 내용을 기반으로 발표.(매 스터디 시작 직전, 사다리타기로 2명 선정)
- 각자 공유한 내용을 기반으로 토론 주제 선정 및 자유 토론 30분 진행
- 모든 인원은 학습 도중 새롭게 알게 된 키워드 공유. (책 내용과 연관된 기업 기술블로그 혹은 특정 키워드에 대해 소개 및 공유)
